# Build Phases

## Phase 1: Project Setup

- Create the Next.js app.
- Add Convex.
- Set up environment variables.
- Create the first interviewer console route.
- Add basic styling and layout.

## Phase 2: Interview Sessions

- Create interview records in Convex.
- Add start interview and stop interview actions.
- Track session status.
- Show current session state in the UI.

## Phase 3: Live Transcription

- Request browser microphone permission.
- Stream audio to the transcription provider.
- Display partial transcript locally.
- Save final transcript segments to Convex.
- Support English, Swahili, and mixed speech.

## Phase 4: AI Next Questions

- Build the Convex action that calls the AI model.
- Send recent transcript and session context.
- Generate 3-5 next questions.
- Save suggestions to Convex.
- Display suggestions in real time.

## Phase 5: Interviewer Controls

- Mark a question as asked.
- Dismiss weak questions.
- Save important questions.
- Regenerate questions manually.
- Add an important moment bookmark.
- Switch interview mode.

## Phase 6: Review and Export

- Show full transcript after the session.
- Show all asked and suggested questions.
- Export transcript as text or markdown.
- Export question list.

## Phase 7: Later Product Expansion

- Audio recording backup.
- Video support.
- Speaker diarization.
- Topic timeline.
- Post-interview analysis.
- Family archive portal.
- Long-term preservation package.
